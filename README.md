# UIUC_ECE120_LAB


UIUC ECE120: Introduction to Computing

This repository contains my personal solutions, lab reports, and homework assignments for UIUC's ECE120 (Introduction to Computing). It includes all completed handworks, code implementations, and problem set answers from the course.

Contents:

Homework: Solutions to assigned problem sets.
Labs: Code, reports, and hands-on assignments from weekly lab sessions.
Projects/MPs: Implementations of larger machine problems.
Purpose:
This repository serves as a personal archive of my work and a demonstration of my learning journey through the fundamental concepts of computing, digital logic, and machine organization.

Important Note:
This code is provided for reference and educational purposes only. If you are currently taking ECE120, please adhere to the University of Illinois Academic Integrity policies. Use this resource to understand concepts and overcome challenges, but always submit your own original work.
